# N-edu Admin Frontend

Draft version of the N-edu Admin Panel.

## Features
- **Dashboard**: Overview statistics (Mock data).
- **User Management**: List, search, view details, and edit users (Mock data).
- **Responsive Design**: Works on Desktop, Tablet, and Mobile.
- **UI Components**: Built with customizable shadcn/ui components.

## Tech Stack
- Next.js 14 (App Router)
- TailwindCSS
- Lucide React Icons

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.
